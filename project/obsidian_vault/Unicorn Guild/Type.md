# Unicorn Guild — Type

A literary zine, not a bank.

---

## Families

| Family | Use | Source |
|---|---|---|
| **Fraunces** | Headlines (humanist serif, optical sizes) | Google Fonts |
| **Figtree** | Body, UI, conversational copy | Google Fonts |
| **Caveat** | Handwritten margin notes (use ONCE per section, max) | Google Fonts |

## Scale

| Style | Spec | Example use |
|---|---|---|
| H1 / Display | Fraunces 500 · 64/66 · opsz 144 · −0.015em | Hero headlines |
| H2 / Section | Fraunces 500 · 42/46 · opsz 72 | Section openers |
| H3 / Sub | Fraunces 500 · 28/32 · opsz 36 | Article subheads |
| Lede | Figtree 400 · 22/32 | Opening paragraph |
| Body | Figtree 400 · 17/27 · max 540px | Default running text |
| Caption | Figtree 500 · 13/19 | Photo captions |
| Label / eyebrow | Figtree 700 · 11.5px · 0.22em UPPERCASE | Section labels |
| Handwritten | Caveat 600 · 32/36 | Margin notes only |

## Italic

Italic body uses **Plum** (`#7A3E5E`) for emphasis — gives Spanish asides and editorial pull-outs a literary feel without shouting.

## Banned

- Helvetica, Arial, Roboto, Inter (too generic, too SaaS)
- All-caps body text
- Letter-spacing on body copy
- Drop shadows on type
